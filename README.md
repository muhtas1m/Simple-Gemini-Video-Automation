# Simple-Gemini-Video-Automation
This is a video automation I made to create videos by 1 click. Just put your title and a short description of what you want to create.

It's very simple to use. Just follow the instructions.

Step 1: Download this repository or clone
Step 2: Make sure you have python installed
Step 3: pip install -q -U google-genai
Step 4: Run the .bat file and enjoy
